# snowsquad

CityRide Analytics Dashboard - Streamlit app connected to Snowflake.

## Overview

This app visualizes key metrics for CityRide, including total revenue, average ride duration, and total rides. Data is fetched from Snowflake and displayed in a clean, interactive Streamlit dashboard.

## Requirements

- Python 3.x  
- Streamlit 1.32.0  
- Snowflake connector 3.6.0  
- Pandas 2.2.0  

Install locally (optional):

```bash
python -m pip install streamlit==1.32.0
python -m pip install snowflake-connector-python[pandas]==3.6.0
python -m pip install pandas==2.2.0
